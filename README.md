# Customer Purchase Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow using Python, MySQL, and Power BI. The objective is to analyze customer purchasing behavior, clean and transform raw data, perform SQL-based analysis, and build an interactive dashboard that provides meaningful business insights.

The project showcases essential data analytics skills, including data preprocessing, exploratory data analysis (EDA), database management, SQL querying, data visualization, and reporting.

---

## Dataset

The dataset contains customer purchase information with features such as:

* Customer ID
* Age
* Gender
* Item Purchased
* Category
* Purchase Amount
* Location
* Size
* Color
* Season
* Review Rating
* Subscription Status
* Shipping Type
* Discount Applied
* Promo Code Used
* Previous Purchases
* Payment Method
* Frequency of Purchases
* Age Group
* Purchase Frequency (Days)

## Project Workflow

### 1. Data Loading

* Imported the customer dataset into Python.
* Examined the dataset structure and data types.

### 2. Exploratory Data Analysis (EDA)

* Checked missing values
* Generated descriptive statistics
* Identified data distributions
* Analyzed customer demographics
* Explored purchasing patterns
* Visualized important trends using charts

### 3. Data Cleaning

* Removed duplicate records
* Corrected inconsistent values
* Standardized column names
* Created additional analytical features
* Exported the cleaned dataset as CSV

### 4. SQL Analysis (MySQL)

* Created a database and customer table
* Imported the cleaned dataset into MySQL
* Executed SQL queries to answer business questions, including:

  * Total sales
  * Average purchase amount
  * Top-selling product categories
  * Customer distribution by location
  * Seasonal purchasing trends
  * Payment method analysis
  * Subscription analysis
  * Customer segmentation

### 5. Power BI Dashboard

Built an interactive dashboard including:

* Sales Overview
* Customer Demographics
* Product Category Analysis
* Seasonal Sales Trends
* Payment Method Distribution
* Location-wise Purchases
* Subscription Status Analysis
* Interactive Filters and Slicers

---

## Key Results

Some important insights obtained from the analysis include:

* Identified the highest-performing product categories.
* Analyzed customer purchasing behavior across different age groups.
* Compared purchase trends by season and location.
* Evaluated the impact of discounts and promotional codes.
* Identified popular payment methods.
* Built an interactive dashboard to support business decision-making.
